Auto Car Dealer is the HTML template which is built with the bootstrap framework (version: 3.3.6). Auto Car Dealer template provides two different layouts of HOME page. It contains 17 individual HTML pages with with 100% responsive & W3C HTML validate coding. All HTML & CSS codes are commented properly so it's easily editable.

File Structure

The file structure of the template is really easy. In the template's root directory contains HTML files and CSS, JS, FONTS, IMG folder. All the stylesheet contained into CSS folder, all the JS files contained in JS folder, all the images contained into IMG folder. So all assets are placed in separate places.

From the above screenshot, you can see the file structure. It's quite easy, right?

 

Navigations

Auto Car Dealer has a nice navigation option. It's allowed to add a menu with sub-menus. It has an individual dropdown of each menu to manipulate its submenus. 

Add New Menu

To add menu without submenu just add <li>...</li> inside navigation ul. And to add menu with submenu you have to add a <li>..</li>  with 'dropdown class', and then add another ul/li inside created 'li'

 

 

 

Logo

The logo contains in almost every page except signup, login, forgot password & 404 page.

To change the logo just put your logo anywhere inside template directory and then just replace the image source. see the screenshot here img/logo.png is the current logo source. Just have to change that source to change the logo

 

 

 

Banner

Banner contains in only two page (Home and Home 2 page).

See this screenshot, from here you can manage your banner content. You can change your banner content (TEXT), add more banner from this area.

And this screenshot for changing the banner background. Just replace the background image's URL path in .banner .item-1, .banner .item-2, .banner .item-3

New Car (Grid Page)

This template has two different grid layout (grid with left sidebar, grid with right sidebar).

To add a new car in grid layout just copy this area and paste. Then modify the contents for the new car.

 

 

New Car (List Page)

This template has two different list layout (list with left sidebar, list with right sidebar).

 

To add a new car in list layout just copy this area and paste. Then modify the contents for the new car.

Sidebar (Search Section)

Sidebar (Car Search Section) is for searching the car listing. It's contained in four pages (car list, car list left sidebar, car grid, car grid left sidebar).

From this section you can manage your car search parameter (Fuel, Transmission, Category, Brand etc)

Car Details Carousel (Slideshow)

Car details page has a bootstrap default carousel for showing car images.

To add more slider image just copy the item div and paste.

Contact Form

Contact form contained in the contact page. It's for contacting with dealer.

From this section, you can manage your contact section. Here you can add/remove more field.

 

Footer

The footer contains in almost every page except signup, login, forgot password, 404 page.

From this section your can manage your template's footer area.
