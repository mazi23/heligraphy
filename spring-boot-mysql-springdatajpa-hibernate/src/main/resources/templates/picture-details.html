Wie kann ich bezahlen?

Es besteht die Möglichkeit per Vorkasse oder Onlinebanking zu bezahlen.

Dürfen von meinem Haus Bilder aufgenommen werden?

Laut §130 des österreichischen Luftfahrtgesetzes, Stand 22.09.2015, dürfen Luftaufnahmen durchgeführt werden. Bilder dürfen aber nicht einfach öffentlich bereit gestellt werden. Deshalb sind unsere Bilder mit einem zufällig erstellten Code hinterlegt. Dieser Code wird an den Besitzer des fotografierten Hauses zugesandt. Somit können die Bilder nur von jenen eingesehen werden die diesen Code besitzen.

Wie komme ich zu einem Bild?

Sobald wir ein Bild für Sie gemacht haben, erhalten Sie eine Postkarte mit einem persönlichen Code. Mit diesem Code können Sie auf unserer Startseite Ihre Bilder einsehen. Wenn Ihnen ein Bild gefällt, können Sie das gewünschte Paket auswählen und das Bild in den Einkaufswagen legen. Dann werden noch Ihre Adressdaten benötigt. Bevor Sie die Bestellung abschließen, können Sie nochmals Ihren Einkauf überprüfen. Wenn Sie auf den Button "Zahlungspflichtig bestellen" drücken, ist Ihre Bestellung abgeschlossen und Sie erhalten eine Bestätigungsmail.

Sind Versandkosten zu zahlen?

Wir berechnen eine Versandpauschale von 5€ bei Zahlung per Vorkasse und 8€ bei Zahlung mit Nachnahme. Für einen Download verrechnen wir keine weiteren Kosten.

Wie lange dauert der Versand?

Rechnen Sie mit etwa 7 Werktagen. In den meisten Fällen kommt das Paktet aber früher.
